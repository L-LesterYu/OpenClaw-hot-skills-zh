# 智能会议纪要生成器

你花大量时间在会议中，做笔记，试图捕捉所有重要信息。等到会议结束时，你已经有几页笔记需要整理、总结和提取行动项。而下一场会议30分钟后就要开始。

这个用例将OpenClaw变成智能会议助手，能够参加你的会议，做智能笔记，并自动生成可操作的摘要。

## 痛点

传统会议笔记记录存在几个问题：
- 手动做笔记会分散注意力，让你无法专注于对话
- 笔记往往不完整或杂乱无章
- 提取行动项和决定很耗时
- 会议洞察在讨论的噪音中丢失
- 跟进行动项成为一项单独的任务

## 它能做什么

- **自动化笔记记录**：OpenClaw参加你的会议（线上或线下）并捕捉关键点、决定和行动项
- **智能摘要**：生成聚焦于结果和关键要点的简洁摘要
- **行动项提取**：自动识别并分类行动项，包含负责人和截止日期
- **会议分析**：提供会议效率、发言时间分布和情感分析的洞察
- **自动化跟进**：为行动项创建日历事件并发送提醒邮件

## 所需技能

- 日历集成（Google Calendar、Outlook等）
- 邮件集成用于跟进
- `web_fetch`用于访问会议记录（如需要）
- `exec`用于本地会议录制工具
- 可选：`browser`用于虚拟会议平台

## How To Set Up

### 1. 基础配置

设置OpenClaw与你的日历和邮件集成。创建AGENTS.md配置：

以下提示词定义了会议笔记助手的角色和行为：

```text
## 会议笔记助手

你是我的智能会议助手。你的工作是参加会议，做智能笔记，并生成可操作的摘要。

访问权限：
- 我的日历（读取权限）
- 我的邮件（发送权限用于跟进）
- 会议录制/记录（如可用）

行为规则：
- 专注于捕捉做出的决定和分配的行动项
- 记录每个行动项的负责人和截止日期
- 用3-5个要点总结关键要点
- 标记任何未解决的问题或需要的跟进
- 在会议结束后1小时内发送跟进邮件
```

### 2. 会议参会设置

对于虚拟会议，设置对会议平台的访问：

以下提示词让智能体处理虚拟会议：

```text
对于虚拟会议（Zoom、Google Meet、Microsoft Teams）：
- 当我加入会议时，如果允许则自动开始录制
- 使用可用工具捕获音频转录
- 对于Zoom会议，使用Zoom API获取转录
- 对于Google Meet，使用官方会议转录功能
```

### 3. 本地会议录制设置

对于线下会议或平台录制不可用时的情况：

以下提示词处理本地会议录制：

```text
对于本地会议录制：
- 使用`exec`访问本地音频录制工具
- 使用Whisper或类似转录服务将音频转换为文本
- 处理转录以提取关键信息
- 从处理后的转录生成笔记和行动项
```

### 4. 每日会议摘要

设置每日cron作业来审查会议：

以下提示词安排每日会议摘要：

```text
每天下午5:00：
- 审查过去24小时的所有会议
- 生成明天到期的行动项摘要
- 发送会议结果和下一步的简要报告
- 标记任何超时或参与度低的会议
```

### 5. 会议质量洞察

以下提示词提供会议质量分析：

```text
对于每个参加的会议，提供：
- 会议时长与计划时间的对比
- 做出的关键决定（带时间戳）
- 带有负责人和截止日期的行动项
- 讨论的情感分析
- 发言时间分布（谁发言最多/最少）
- 下次更有效会议的建议
```

## 关键洞察

- **AI不会分心**：与人类不同，AI能够关注每个细节而不会遗漏重要点
- **自动化跟进**：最大的价值是创造问责制 - AI自动发送跟进邮件和日历事件
- **会议智能**：分析会议模式随时间变化，揭示团队动态和会议效率的洞察
- **实时价值**：AI可以在会议期间提供实时笔记供参与者参考

## 示例提示词

**开始会议参会：**
> "我要参加一个关于Q2计划的销售团队会议。请参加会议并记录决定和行动项。"

**处理会议录制：**
> "我在/Users/username/Documents/meeting-recording.wav有一个会议录制。请转录它并生成包含行动项的会议笔记。"

**生成每周摘要：**
> "创建所有会议的每周摘要，显示行动项进展和每个讨论的关键结果。"

**会议效率分析：**
> "分析我本周的会议模式 - 哪些会议最有效率，我应该改变什么模式？"

## 实用建议

- **用示例会议测试**：在重要会议之前用几个会议测试系统
- **自定义行动项类别**：根据团队工作流调整类别
- **设置正确权限**：确保日历和邮件权限正确配置
- **使用一致的会议格式**：标准会议结构更适合AI分析
- **定期审查**：审查AI生成的笔记准确性，并根据需要调整提示词
