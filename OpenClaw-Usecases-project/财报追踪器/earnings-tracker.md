# AI 驱动的财报追踪器

追踪数十家科技公司的财报季节意味着需要查看多个来源并记住报告日期。你想在不手动追踪每家公司的情况下，随时掌握 AI/科技公司的财报动态。

这个工作流程自动化了财报追踪和推送：

• 每周日预览：扫描即将到来的财报日程，将相关科技/AI公司发布到 Telegram
• 你选择关心的公司，OpenClaw 为每个财报日期安排一次性 cron 任务
• 每份财报发布后，OpenClaw 搜索结果，格式化详细摘要（超出/未达预期、关键指标、AI 亮点），并推送给你

## 所需技能

- `web_search`（内置）
- OpenClaw 的 Cron 任务支持
- 用于财报更新的 Telegram 频道

## 如何设置

1. 创建一个名为 "earnings" 的 Telegram 频道用于更新。
2. 提示 OpenClaw：
```text
每周日晚上 6 点，运行一个 cron 任务：
1. 搜索接下来一周科技和 AI 公司的财报日程
2. 筛选我关心的公司（NVDA、MSFT、GOOGL、META、AMZN、TSLA、AMD 等）
3. 将列表发布到我的 Telegram "earnings" 频道
4. 等待我确认要追踪哪些公司

当我回复要追踪哪些公司时：
1. 为每个财报日期/时间安排一次性 cron 任务
2. 每份财报发布后，搜索财报结果
3. 格式化摘要，包括：超出/未达预期、营收、每股收益、关键指标、AI 相关亮点、展望
4. 发布到 Telegram "earnings" 频道

记录我通常追踪的公司，以便每周自动建议。
```
