# 多智能体内容工厂

你是一名内容创作者，每天都在多个平台间协调研究、写作和设计。每个步骤——寻找热门话题、撰写脚本、生成缩略图——都会花费你数小时的时间。如果有一支专业的智能体团队在一夜之间处理所有这些事情，会怎样？

这个工作流程在Discord内部建立了一个多智能体内容工厂，不同的智能体在专门的频道中处理研究、写作和视觉资源。

## 功能特点

- **研究智能体**每天早上扫描热门故事、竞争对手内容和社交媒体，寻找最佳的内容机会
- **写作智能体**采用最佳想法，撰写完整的脚本、帖子或通讯草稿
- **缩略图智能体**为内容生成AI缩略图或封面图片
- 每个智能体在自己的Discord频道中工作，保持一切井井有条且可审核
- 按计划自动运行（例如每天上午8点），让你醒来时就能看到完成的内容

## 痛点

内容创作有三个阶段——研究、写作和设计——大多数创作者都在手动完成所有这些工作。即使有AI写作工具，你仍然必须一个接一个地提示它们。这个系统将智能体连接成一个流水线，一个智能体的输出成为下一个智能体的输入，完全无需人工干预。

## 所需技能

- Discord多频道集成
- `sessions_spawn` / `sessions_send` 用于多智能体编排
- [x-research-v2](https://clawhub.ai) 或类似工具用于社交媒体研究
- 本地图像生成（如Nano Banana）或图像生成API
- [knowledge-base](https://clawhub.ai) 技能（可选，用于RAG驱动的研究）

## 如何设置

1. 设置Discord服务器（或者让OpenClaw为你设置——只需说"为我们设置一个Discord"）。

2. 为每个智能体创建频道：
   - `#research` ——热门话题和内容机会
   - `#scripts` ——写作草稿和大纲
   - `#thumbnails` ——生成的图片和封面艺术

3. 提示OpenClaw：
```text
我想要在Discord内部建立一个内容工厂。
为不同的智能体设置频道：

1. 研究智能体（#research）：每天早上8点，研究我所在领域的热门
   故事、竞争对手内容和社交媒体上的表现良好的内容。
   发布前5个内容机会及来源。

2. 写作智能体（#scripts）：从研究智能体的最佳想法中
   选择一个，撰写完整的脚本/帖子/通讯草稿。发布到#scripts。

3. 缩略图智能体（#thumbnails）：为内容生成AI缩略图或封面图片。
   发布到#thumbnails。

让他们的工作在不同的频道中有条理地组织。
每天早上自动运行这个流水线。
```

4. 根据你的平台进行定制：
```text
我专注于X/Twitter帖子，而不是YouTube。将写作智能体
改为生成Twitter帖子而不是视频脚本。
```

## 关键洞察

- **威力在于连锁智能体**——研究驱动写作，写作驱动缩略图。你不需要单独提示每个步骤。
- Discord频道让你可以分别审核每个智能体的工作，并提供反馈，比如"脚本太长"或"多关注AI新闻"。
- 你可以将此适配到任何内容格式：推文、通讯、LinkedIn帖子、播客大纲、博客文章。
- 在本地运行图像生成模型（如Mac Studio上的Nano Banana）可以降低成本并让你更好地控制。
