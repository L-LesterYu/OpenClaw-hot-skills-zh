# 通过聊天完成AI视频编辑

通常的视频编辑意味着打开时间线编辑器，拖拽片段，点击菜单。对于重复的编辑工作——修剪片头，为批量片段添加字幕，调整10个视频的颜色——这个手动循环会消耗大量时间。

这个用例将视频编辑转化为对话。描述你想要更改的内容，直接发送文件，然后获得处理结果。无需时间线，无需GUI。

## 你可以做到的

- 通过自然语言描述时间戳来修剪、剪切和合并片段
- 添加带自动音频压制的背景音乐
- 从语音生成并烧录字幕（支持50+种语言）
- 调整视频色调（"让它更暖"、"匹配第一个片段的视觉效果"）
- 裁剪为竖版格式，适用于TikTok/Reels/Shorts
- 使用相同的编辑方案批量处理多个文件

## 你需要的技能

- [video-editor-ai](https://clawhub.ai/skills/video-editor-ai) — 基于聊天界面的视频编辑，支持背景音乐、字幕和导出
- [ai-subtitle-generator](https://clawhub.ai/skills/ai-subtitle-generator) — 自动生成字幕、字幕烧录、SRT导出

## 如何设置

1. 安装技能：
```bash
clawhub install video-editor-ai
clawhub install ai-subtitle-generator
```

2. 将视频文件发送到聊天中，描述你的编辑需求：
```text
将这个视频从0:15修剪到1:30，添加背景音乐（活泼一点的），并烧录英文字幕。
```

3. 对于批量处理，描述处理模式：
```text
我在 /videos/raw/ 目录下有5个片段。对每个片段：
- 裁剪为9:16竖版
- 在底部添加自动生成的字幕
- 导出为mp4格式
```

智能体会处理API调用，轮询完成状态，并将处理好的文件发送回聊天。

## 使用技巧

- 明确指定时间戳和输出格式（"导出为1080p的mp4"）
- 字幕工作如果源语言不是英语，请说明源语言
- 调色时，参考描述性语言（"温暖夕阳色调"）比技术参数效果更好
- 避免上传敏感内容（人脸、身份证件、机密屏幕画面），除非你已审查过服务商的数据保留和隐私政策
