# OpenClaw Usecases 中文版

这是 OpenClaw Usecases 的中文翻译版本，收录了各种实用的 OpenClaw 应用场景。

## 已翻译的用例列表（28 个）

### 📋 任务与项目管理
- **会议纪要与待办事项自动化** (`meeting-notes-action-items`) - 自动生成会议纪要、提取行动项并录入 Jira/Linear/待办清单
- **Todoist 任务管理器** (`todoist-task-manager`) - 将智能体推理和进度日志同步到 Todoist，最大化工作流透明度
- **项目状态管理系统** (`project-state-management`) - 事件驱动的项目状态追踪方案，替代传统看板
- **目标驱动的自主任务系统** (`overnight-mini-app-builder`) - AI 智能体了解你的目标并每天主动提出推进任务
- **自愈式家庭服务器** (`self-healing-home-server`) - AI 自动监控和修复家庭服务器基础设施问题

### 💬 通信与通知
- **电话通知** (`phone-call-notifications`) - 让智能体通过电话通知紧急事项，支持双向实时对话
- **多渠道个人助手** (`multi-channel-assistant`) - 统一接口连接所有工具，跨应用管理任务和消息
- **活动嘉宾确认** (`event-guest-confirmation`) - 使用 SuperCall 逐一拨打嘉宾电话确认出席情况

### 📱 社交媒体
- **X/Twitter 自动化操作** (`x-twitter-automation`) - 通过自然语言实现完整 X/Twitter 自动化
- **X 账号分析** (`x-account-analysis`) - 提供 X 账号的深度定性分析
- **YouTube 内容管线** (`youtube-content-pipeline`) - 自动追踪网络和 X 上的视频创意
- **每日 YouTube 摘要** (`daily-youtube-digest`) - 每天 YouTube 频道新视频个性化摘要

### 🎥 内容创作
- **通过聊天完成 AI 视频编辑** (`ai-video-editing`) - 将视频编辑转化为对话，支持修剪、字幕、调色和批量处理

### 🔬 研究与知识
- **Hugging Face 论文研究发现** (`hf-papers-research-discovery`) - 对话式发现、筛选和深度阅读 HF 热门论文
- **个人知识库（RAG）** (`knowledge-base-rag`) - 从保存的所有内容中构建可搜索的知识库
- **第二大脑** (`second-brain`) - 像给朋友发短信一样简单地捕获想法和内容

### 📊 市场与商业
- **市场调研与产品工厂** (`market-research-product-factory`) - 从 Reddit 和 X 上挖掘真实用户痛点
- **多渠道 AI 客户服务平台** (`multi-channel-customer-service`) - 同时处理多平台客户消息
- **个人 CRM 与自动联系人发现** (`personal-crm`) - 自动记录人脉关系和跟进事项
- **本地 CRM 框架（DenchClaw）** (`local-crm-framework`) - 一条命令将 OpenClaw 变为完全本地 CRM
- **Polymarket 自动驾驶** (`polymarket-autopilot`) - 自动监控预测市场套利机会

### 📰 信息聚合
- **多源科技新闻摘要** (`multi-source-tech-news-digest`) - 自动聚合 109+ 来源的科技新闻
- **收件箱整理** (`inbox-declutter`) - 自动整理收件箱中的新闻订阅邮件
- **自定义早间简报** (`custom-morning-brief`) - 每天定时推送完全定制的早间报告

### 🏠 生活与效率
- **家庭日历聚合与家务助手** (`family-calendar-household-assistant`) - 聚合多平台家庭日历并管理家务
- **健康与症状追踪器** (`health-symptom-tracker`) - 长期持续记录健康症状和饮食模式

### 🔧 开发与运维
- **OpenClaw + Opik 可观测性追踪** (`opik-openclaw-observability`) - 将 OpenClaw 运行过程接入 Opik 实现统一追踪
- **OpenClaw + n8n 工作流编排** (`n8n-workflow-orchestration`) - 让 AI 智能体安全管理 API 密钥和调用外部服务
