# OpenClaw Usecases 中文版

这是 OpenClaw Usecases 的中文翻译版本，收录了各种实用的 OpenClaw 应用场景。

## 已支持的技能列表

### 自动化工作流
- **自定义早间简报** (`custom-morning-brief`) - 每天定时推送完全定制的早间报告，涵盖新闻、任务、创意和主动建议
- **OpenClaw + Opik 可观测性追踪** (`opik-openclaw-observability`) - 将 OpenClaw 运行过程接入 Opik，实现统一追踪、错误排查和成本监控
