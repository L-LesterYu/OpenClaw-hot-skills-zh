# 🖥️ 编程助手启动包

面向软件开发预配置的 AI 人格设置。

---

## 包含内容

| 文件 | 用途 |
|------|------|
| `SOUL.md` | Axiom — 直接、技术导向的编程助手 |
| `HEARTBEAT.md` | 每日运维，包含 CI/CD 检查、PR 审查、构建状态 |
| `KNOWLEDGE.md` | 技术栈参考、代码模式、常见问题 |

---

## 使用方法

1. 将这些文件复制到您的工作区
2. 针对您的技术栈进行自定义：
   - 更新 KNOWLEDGE.md 中的语言/框架
   - 添加您仓库的模式和约定
   - 配置您的 CI/CD 和部署信息
3. 运行设置向导生成其余文件（USER.md、MEMORY.md 等）

---

## 适合以下用户：

- 开发者或拥有开发团队
- 希望获得编码、调试和代码审查方面的帮助
- 使用基于 Git 的工作流
- 有需要监控的 CI/CD 管道

---

## 需要自定义的内容：

### SOUL.md 中：
- [ ] 将 "Alex" 更改为您的名字
- [ ] 更新技术偏好
- [ ] 调整沟通风格

### HEARTBEAT.md 中：
- [ ] 添加您实际的仓库和 PR 链接
- [ ] 配置您的 CI/CD 命令
- [ ] 设置监控告警

### KNOWLEDGE.md 中：
- [ ] 替换为您自己的技术栈
- [ ] 添加您的代码库模式
- [ ] 记录您的部署流程
- [ ] 添加环境变量
