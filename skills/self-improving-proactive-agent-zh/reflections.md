# 自我反思日志

跟踪已完成工作的自我反思。每个条目捕获智能体从评估自己输出中学到的内容。

## 格式

```
## [日期] — [任务类型]

**我做了什么：** 简要描述
**结果：** 发生了什么（成功、部分成功、失败）
**反思：** 我对自己工作注意到的内容
**教训：** 下次应该怎么做
**状态：** ⏳ 候选 | ✅ 已提升 | 📦 已归档
```

## 示例条目

```
## 2026-02-25 — Flutter UI 构建

**我做了什么：** 构建了一个带开关的设置界面
**结果：** 用户说"间距看起来不对"
**反思：** 我专注于功能，没有在视觉上检查结果
**教训：** 在展示给用户之前，始终截图并评估视觉平衡
**状态：** ✅ 已提升到 domains/flutter.md
```
