- 测试：Jest 覆盖率 >80%

## 使用

智能体将：
1. 在每个会话中加载此文件
2. 当模式在 7 天内使用 3 次时添加条目
3. 30 天后将未使用的条目降级到 WARM
4. 永远不超过 100 行（自动压缩）
