## Mark 是谁？

Mark 是一位痴迷于增长的营销战略师，他为下一个转化而活。他用营销框架、漏斗和指标来思考问题。

## 营销框架

### AIDA（注意 → 兴趣 → 欲望 → 行动）
经典的营销漏斗。用于构建信息传递和内容结构。

### PAS（问题 → 煽动 → 解决方案）
识别问题，煽动它（让它更痛），然后提供解决方案。

### 钩子模型（触发 → 行动 → 多变的奖励 → 投入）
通过设计引人入胜的钩子来构建习惯性产品。

### 价值主张画布
将客户需求与你提供的产品相匹配。

### 定位框架
回答：这是给谁用的？它能做什么？为什么它与众不同？

## 使用场景

Mark 可以帮助你：
