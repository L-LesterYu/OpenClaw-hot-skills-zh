# 公文格式化技能

按照中国党政机关公文格式标准（GB/T 9704-2012），对文档进行格式化处理。

## 一、页面设置

| 项目 | 正式公文 | 非正式公文 |
|------|----------|------------|
| 上边距 | 3.7 厘米 | 3.7 厘米 |
| 下边距 | 3.2 厘米 | 3.2 厘米 |
| 左边距 | 2.8 厘米 | 2.8 厘米 |
| 右边距 | 2.6 厘米 | 2.6 厘米 |
| 页眉 | 1.5 厘米 | 1.5 厘米 |
| 页脚 | 2.6 厘米 | 2.6 厘米 |

## 二、红头（仅正式公文）

正式公文需套单位红头，红头内容为发文机关全称，如：

```
×××××××××××文件
```

## 三、正文格式

### 3.1 基本设置

- **正文字体**：仿宋_GB2312
- **段落首行缩进**：2个汉字（约2字符）
- **行间距**：固定值 29 磅
- **正文与标题之间**：空一行

### 3.2 标题层级与字体

| 层级 | 格式 | 字体 | 示例 |
|------|------|------|------|
| 一级标题 | 一、二、三、四 | **黑体 3号** | 一、总体要求 |
| 二级标题 | （一）（二）（三） | **楷体_GB2312** | （一）指导思想 |
| 三级标题 | 1. 2. 3. | **仿宋_GB2312** | 1. 完善工作机制 |

### 3.3 阿拉伯数字处理

全文中的阿拉伯数字可统一为以下两种方式之一：

- **方式一**：统一使用仿宋_GB2312，如：12345
- **方式二**：统一使用 Times New Roman，如：12345

> 注意：全文保持一致，不可混用。

## 四、页码格式

| 公文类型 | 格式 | 位置 |
|----------|------|------|
| 正式公文 | —1—（数字前后加横线） | 单页靠右，双页靠左 |
| 非正式公文 | 1（纯数字） | 居中 |

## 五、落款格式

### 5.1 落款单位名称

| 公文类型 | 单位名称 | 示例 |
|----------|----------|------|
| 正式公文 | 全称 | 宁波市数据局 |
| 非正式公文 | 可用简称 | 市数据局 |

- **位置**：靠右，适当空几个字（美观即可，不必到底）
- **与正文间距**：空 2 行

### 5.2 落款日期

- **格式**：阿拉伯数字，如 `2026年3月13日`
- **位置**：对应上方落款单位名称居中

**示例：**

```
                    宁波市数据局

                        2026年3月13日
```

## 六、附件格式

### 6.1 附件标注位置

- 附件与正文之间空 **1 行**
- 多个附件的标题与第1个标题位置对齐
- 附件标题较长时，另起一行与上一行首字对齐

### 6.2 附件标签

- **"附件"二字**：黑体 3号，位于左上角

**示例：**

```
附件：

1. ×××××××实施方案
2. ×××××××任务分工表
```

## 七、完整格式模板（Markdown → Word 转换参考）

```markdown
# ×××××关于×××××的通知

（正文与标题空一行）

  各区县（市）人民政府，市政府各部门、各直属单位：

  （正文内容，仿宋_GB2312，首行缩进2字符，行间距29磅）……

  一、一级标题（黑体3号）

  （一）二级标题（楷体_GB2312）

  1. 三级标题（仿宋_GB2312）

  正文内容……

  
                    落款单位全称（空2行后，靠右）

                        2026年3月13日（居中）


附件：（空1行）

1. 附件标题一
2. 附件标题二
```

## 八、Markdown 输出格式约定

当使用此技能格式化文档时，按以下约定输出：

```markdown
> **格式说明**
> - 一级标题（一、二、三、四）：黑体3号
> - 二级标题（（一）（二）...）：楷体_GB2312
> - 三级标题（1. 2. 3...）：仿宋_GB2312
> - 正文：仿宋_GB2312，段落前空2个汉字
> - 行间距：固定值29磅
> - 页边距：上3.7cm、下3.2cm、左2.8cm、右2.6cm
> - 页眉1.5cm、页脚2.6cm
> - 页码：数字前后加横线（—1—），单页靠右，双页靠左

---

## 一、一级标题

### （一）二级标题

1. 三级标题

  正文内容（注意前面有2个空格缩进）...

```

## 九、常见公文类型

| 类型 | 特点 | 示例标题 |
|------|------|----------|
| 通知 | 传达事项、部署工作 | 关于×××的通知 |
| 通报 | 表彰先进、批评错误 | 关于×××情况的通报 |
| 决定 | 重要事项、重大行动 | 关于×××的决定 |
| 意见 | 提出见解、办法 | 关于×××的实施意见 |
| 方案 | 具体工作安排 | ×××实施方案 |
| 报告 | 汇报工作、反映情况 | 关于×××情况的报告 |
| 请示 | 请求指示、批准 | 关于×××的请示 |
| 函 | 商洽工作、询问答复 | 关于×××的函 |

## 十、注意事项

1. **字体一致性**：全文字体风格保持统一，不可混用
2. **层级清晰**：标题层级一般不超过三级，层级过多时可使用附件
3. **数字规范**：公文中的数字一般使用阿拉伯数字，成文日期用阿拉伯数字
4. **标点规范**：标点符号使用中文全角标点
5. **引文规范**：引用公文应注明发文机关、标题、发文字号

## 十一、Word 格式化快捷操作

转换为 Word 文档时的快捷设置：

1. **页面设置**：页面布局 → 页边距 → 自定义边距
2. **行间距**：段落 → 行距 → 固定值 → 29磅
3. **首行缩进**：段落 → 特殊格式 → 首行缩进 → 2字符
4. **字体设置**：选中文字 → 字体对话框 → 选择对应字体
